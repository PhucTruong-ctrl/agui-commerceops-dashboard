# SEO Audit

Dùng khi cần đánh giá sức khỏe SEO của site hoặc landing page.

## Mục tiêu

- Tìm lỗi crawl/indexation, on-page, content quality.
- Ưu tiên hành động theo mức ảnh hưởng (High/Medium/Low).

## Framework audit

1. Crawlability & indexation: robots, sitemap, canonical, noindex, redirect.
2. Technical: Core Web Vitals, mobile usability, HTTPS.
3. On-page: title/meta/H1, internal link, keyword intent.
4. Content & E-E-A-T: độ sâu nội dung, tín hiệu tin cậy.

## Lưu ý quan trọng

- Không kết luận "không có schema" chỉ dựa vào fetch HTML tĩnh.
- Với schema JSON-LD, nên kiểm tra bằng render browser hoặc Rich Results Test.
